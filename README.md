# Deep Learning for NIRS Research

Master's research project investigating deep learning models for the estimation of fruit attributes using near infrared 
spectroscopy.


### Author's Note

Inspired by [dario-passos/DeepLearning_for_VIS-NIR_Spectra](https://github.com/dario-passos/DeepLearning_for_VIS-NIR_Spectra), 
I decided to create a public repository for parts of my master's research project with the [Non-Invasive Sensor Group at CQUniversity's 
Institute for Future Farming Systems](https://www.cqu.edu.au/research/organisations/institute-for-future-farming-systems/non-invasive-sensor-technology). 
The working title of my thesis is: 

**Deep Learning In Estimation Of Fruit Attributes Using Near Infrared Spectroscopy**

Near infrared spectroscopy is used as a non-invasive method for estimating attributes of individual fruit. The technique 
relies on statistic models (‘chemometrics’) to deduce information on sample attributes from spectra collected from the 
fruit. Barriers to the adoption of this technique for practical use in the fruit industry include the robustness of 
models across fruit from different growing conditions and spectra collected on different instruments. My research is 
intended to reduce these barriers to by exploring deep learning techniques for developing robust, global models across 
the season, growing conditions, fruit variety, individual instruments and other variations.

## Set-Up
In a Python 3.8 environment, navigate to this directory and install the requirements:
```bash
pip -r requirements.txt
```
Start running a jupyter server using the following command:
```bash
jupyter notebook
```
This will automatically open a web browser to the URL of the Jupyter notebook web application. From here explore the 
notebooks within this repository and run them.
